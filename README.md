# Student Portfolio

## M1FA1 Portfolio Task by Rabara, Niel

### Project Description
A responsive student portfolio website showcasing achievements, projects, and contact information with dark/light theme toggle functionality.

### Features
-  Responsive design that works on all devices
-  Dark/Light theme toggle
-  Contact information popup
-  Project showcase with images
-  Professional styling
